2. Honesty Gates (Pre-1905 → Post-1905)

Gate 1: Dimensional Purity

Every expression must respect physical dimensions. m+v → TYPE ERROR (kg vs m/s). 100% of candidates must pass.

Gate 2: Era Knowledge Segregation

Training: exclusively pre-1905 domains, symmetries, and scenarios. Testing: all post-1905 physics held out. Configurable via --era-cutoff YEAR flag on every training script. Zero era leakage.

Gate 3: Generalization

Discovered laws must hold on held-out scenarios the system was never trained on. Pre-1905 training → post-1905 test. Score must exceed noise floor at p < 0.05.

Gate 4: Nontriviality

Expression must not be a mathematical identity. depth ≥ 2, contains ≥ 1 dynamic quantity. Reject E-E=0, x/x=1, constant-only expressions.

Gate 5: Mathematical Proof

Numerical constancy must be provable in Lean. The theorem must hold for all initial conditions, not just the observation set. Auto-prover generates proof via tactic search.
